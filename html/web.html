Course Rating and Recommendation System

Description

As a four member team, we designed a course rating and recommendation system for students from school of engineering of Boston University. With our website, you can read reviews of professors and all courses in engineering department. After registration, student could review courses they have ever taken. They will also be given the most popular courses and the system will recommend courses for them.

Neighborhood Map

Description

This neighborhood map is help you to find meetup activities nearby. If you enable your location, the map will be centered in your local place. If not, the default location is Milpitas. The map displays all meetups about coding with 20 miles around your location. You can click eahc item or marker on the map to learn more about it. This project is done using javascript, MVVM framework, knockout.js, jQuery.

My work

1. Fetched data using Ajax and meetup api.
2. Wrote the View part using html.
3. Combined knockout.js to write the ViewModel part.
